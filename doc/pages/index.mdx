# DeepCore Documentation

DeepCore is a Python library for generating synthetic data and processing relationship data. It provides tools for creating realistic synthetic datasets and organizing relationship data in a structured format.

## Key Features

- **Synthetic Data Generation**: Create realistic synthetic data using CTGAN models
- **Relationship Data Processing**: Sort and organize relationship data between entities
- **Easy Integration**: Simple API for data generation and processing tasks

## Installation

```bash
pip install deepcore
```

For local development installation:

```bash
git clone https://github.com/IlyaElevrin/deepcore.git
cd deepcore
pip install -e .
```
